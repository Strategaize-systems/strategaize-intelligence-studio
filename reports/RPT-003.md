# Architecture — StrategAIze Intelligence Studio V1

## Outcome
Architektur vollständig definiert. Alle technischen Entscheidungen getroffen, Datenmodell ausgearbeitet, Komponentenstruktur und Flows dokumentiert.

## Architecture Summary
- **3-Schichten**: UI-Layer (Next.js) → Data-Layer (SQLite) → AI-Layer (Claude CLI)
- **11 Tabellen** + 2 Junction Tables + FTS5 für Suche
- **Server Actions** für alle Mutations
- **Claude CLI** über child_process.execFile
- **DSGVO-Filter** über buildSafePrompt-Mechanismus

## New Decisions
- DEC-006: better-sqlite3 statt ORM
- DEC-007: Claude Code CLI statt API
- DEC-008: Server Actions statt REST API
- DEC-009: Prompt Templates als Flat Markdown Files

## Key Design Choices
- Prompt-Templates in prompts/ als editierbare Markdown-Dateien
- Brand Config als Singleton-Tabelle
- FTS5 für einfache Volltextsuche in V1
- Exports in exports/ Verzeichnis
